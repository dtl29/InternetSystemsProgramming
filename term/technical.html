The architecture of this project was created with php, html, and javascript. Along with a database. In order to create a four tier architecture website
The database schema is made out of three tables:
Decks(name, nameOfCreator, primaryCard, decklist),
DeckUsers(id,username, password, email),
and Cards(name, cmc(converted mama cost), cardType, color, legalBrawl, legalComander... power, rarity, setName, toughness, image).

The Cards table's attributes are all the identifiers for the card within the game to have a better look up for the user, so
the user can look up what card they need easier.
